# 課題管理アプリ

個人用のシンプルな課題管理ウェブアプリです。HTML / CSS / JavaScript のみで構成されており、設定ファイルや外部サービス不要でブラウザから直接利用できます。

## 使い方

`index.html` をブラウザで開くだけで使用できます。

## 機能

- **課題の追加・編集・削除** — タイトル・説明・ステータス・優先度・期限・カテゴリ・タグを管理
- **優先度設定** — 高 / 中 / 低 の3段階（左端のカラーラインで視覚的に識別）
- **ステータス管理** — 未着手 / 進行中 / 完了（チェックボックスでワンクリック切り替え）
- **期限設定** — 期限超過は赤・3日以内は黄色でハイライト
- **カテゴリ・タグ管理** — カテゴリは「カテゴリ管理」から追加・削除、タグはカンマ区切りで自由入力
- **繰り返しタスク** — 毎日・毎週・毎月の繰り返し設定と回数指定で一括生成
- **カレンダー表示** — 月カレンダーで期限日ごとに課題を確認（デフォルト表示）
- **リスト表示** — 課題を一覧で確認。フィルター・検索バーはリスト表示時のみ表示
- **フィルター・検索** — ステータス・優先度・カテゴリで絞り込み、タイトル/説明/タグのテキスト検索（リスト表示専用）
- **データ永続化** — `localStorage` に自動保存（ページを閉じてもデータが残ります）
- **エクスポート・インポート** — カレンダー右下からJSONファイルで書き出し・読み込み（バックアップ・移行に対応）

## キーボードショートカット

| キー | 操作 |
|------|------|
| `n` | 課題追加モーダルを開く |
| `Esc` | モーダルを閉じる |
